# Contagion — Livre de Regles Complet

---

## 1. Creation de Personnage

### Budget de points

A la creation, chaque personnage dispose de **15 points** a repartir librement entre toutes les competences.

> **Contrainte de creation** : Aucune competence ne peut atteindre le palier **Maitre** (7 points cumules) lors de la creation du personnage.

### Schema d'evolution des paliers

| Palier | De | Cout du palier | Cout cumule |
|---|---|---|---|
| Incompetent | d4 | — | 0 |
| Initie | d6 | 1 | 1 |
| Entraine | d8 | 1 | 2 |
| Competent | d10 | 2 | 4 |
| Maitre | d12 | 3 | 7 |
| Legende | d20 | 4 | 11 |

> Le palier **Legende** est reserve a des moments narratifs exceptionnels et ne peut pas etre atteint par allocation de points standard lors de la creation.

### Points bonus du Maitre de Jeu (+i)

Le MJ peut accorder des **points bonus** a une competence specifique, notes `(+i)`. Ces points s'ajoutent au-dela du palier atteint par investissement normal et ameliorent le resultat des jets sans changer de palier.

Exemple : une competence a 5 points investis atteint le palier **Competent** (cout 4), avec 1 point de reste en bonus. Si le MJ ajoute (+1), le bonus total est +2 sur les jets de cette competence.

---

## 2. Statistiques de Base

| Statistique | Valeur de base | Description |
|---|---|---|
| **PS** (Points de Sante) | 10 | Represente la vitalite physique du personnage. A 0, le personnage est en Agonie. |
| **PSM** (Points de Sante Mentale) | 10 | Represente la stabilite mentale. A 0, le personnage sombre dans un etat de crise. |
| **Souffle** | 10 | Ressource depensee pour effectuer des actions en combat. Se recharge entre les rounds. |

Ces valeurs de base peuvent etre augmentees ou diminuees par les talents passifs des competences investies.

## 4. Systeme de Combat

### Le Souffle

Le **Souffle** est la ressource centrale du combat. Chaque action en combat coute un certain nombre de **Charges de Souffle (CS)**. Chaque personnage dispose de 10 Souffle de base (modifiable par les talents).

- Au debut de chaque **Round**, tous les personnages recuperent leur Souffle maximum.
- Le **minimum de CS** pour toute action est de **2**, meme apres application de malus/bonus.
- Les CS depenses pendant un tour representent l'effort physique et mental consenti par le personnage.

### Actions disponibles en combat

| Action | Cout CS | Description |
|---|---|---|
| **ATTAQUE** | 6 CS | Attaquer une cible au corps a corps ou a distance. |
| **ESQUIVE** | 6 CS | Tenter d'eviter une attaque par un mouvement de corps. |
| **PARADE** | 4 CS | Bloquer ou devier une attaque avec son arme ou bouclier. |
| **AIDER** | 6 CS | Soutenir un allie pour ameliorer son prochain jet. |
| **SPECIAL** | 6 CS | Effectuer une action speciale (RECHARGER, TACLER, ATTRAPER, etc.). |

> Le cout de base peut etre modifie par les talents des competences (bonus ou malus de CS).

### Seuil de Toucher (ST)

Le **Seuil de Toucher** de base est de **4+**. Un attaquant doit obtenir au minimum ce resultat sur son jet de competence pour toucher sa cible. Ce seuil peut etre modifie par :
- Les talents passifs (ex : +1 ST, -1 ST)
- La position (EMBUSQUE, ENCERCLE, etc.)
- Les effets speciaux d'armes ou de situations

### Critiques

A partir du palier **Competent**, lorsqu'un personnage obtient un resultat critique (maximum sur le de de competence), il peut declencher un effet superieur selon le contexte : degats bonus, effet etendu, avantage narratif, etc. Le MJ tranche selon la situation.

### Fragilite en PARADE

Lorsqu'un personnage effectue une PARADE, l'arme ou le bouclier utilise peut subir des dommages. Si le resultat du de de l'attaquant correspond a la valeur de **Fragilite** de l'arme (ou en dessous), celle-ci perd 1 point de Fragilite. A 0, l'objet est brise.

## 5. Effets Speciaux

Les effets speciaux s'appliquent aux attaques ou situations decrites. Certains sont lies aux armes, d'autres peuvent etre utilises par les talents de competences.

| Effet | Description |
|---|---|
| **Perforant X** | Ignore X points d'Armure de la cible. |
| **Escrime** | Permet d'utiliser la competence Martial pour les esquives reactives, ou confere un avantage d'initiative contextuel. |
| **Brise-Bouclier** | En cas de touche reussie, l'arme ou le bouclier de la cible perd 1 point de Fragilite supplementaire. |
| **Assommant** | Sur une touche, la cible peut subir l'etat ETOURDI ou ASSOMME selon le contexte, au lieu de degats normaux. |
| **Longue Portee (+1 ST)** | L'arme peut attaquer depuis une plus grande distance. Peut conferer +1 ST dans certains contextes. |
| **Blessant** | Inflige l'etat BLESSE sur une touche reussie. |
| **Entrave** | La cible est ralentie ou immobilisee partiellement sur une touche. |
| **Brulure** | Inflige des degats sur plusieurs rounds (feu ou acide). |
| **Sacre / Profane** | L'arme est particulierement efficace contre des entites d'une nature opposee (mort-vivants, demons, etc.). |
| **Crochet** | Peut desarmer ou faire tomber la cible sur une touche reussie. |
| **Improvisee** | Arme non prevue pour le combat. Penalite possible selon le MJ. |
| **Silencieuse** | L'attaque ne produit pas (ou peu) de son. Bonus contextuel dans la penombre ou pour rester discret. |
| **Fragmentation** | L'effet touche une zone autour de la cible principale. |
| **Hemorrhagie** | Degats continus dus a une plaie ouverte. |
| **Balayage** | Peut toucher plusieurs cibles adjacentes en une seule attaque. |
| **Deflagration** | Provoque une detonation qui peut faire fuir les ennemis faibles ou provoquer la panique. |
| **Disperse** | L'attaque se divise sur plusieurs cibles proches simultanement. |
| **Bris** | Peut briser ou endommager un equipement ou une structure. |
| **Ecrasant** | Force la cible a faire un jet de CORPS pour ne pas tomber ou reculer. |
